Computational Geometry Portfolio and Efficient Database Querying Project

A series of computation geometry projects on Convex Hull algorithms, pathfinding algorithms, and shape/intersections methods, and more using Python. Personal capstone project included implementing n-dimensional Orthogonal Range Querying data structure, a way of querying a database in O(log^(d-1) n + k) time where d is the number of dimensions, n the number of items in the database, and k the number of items returned. This is achieved using a recursive-binary-tree-like data structure with O(n log^(d-1) n) space complexity. Currently working on creating an algorithm to recursively rebalance the data structure when items are added or removed.
